Josh White — CTO @ Alchemize. Solo developer across all three active products (Lead System Backend, Alchemize Portal, Scraper). No other contributors; no need to hedge for team conventions.

**Role context:** Builds and operates the full stack — Java/Spring backend, React frontend, Python scraper, Coolify infrastructure, Jira project management. Makes all architectural and product decisions.

**Working style:**
- Delegates autonomously once direction is set: "you can set up the rest, you can do it all" means proceed without hand-holding.
- Expects one-line updates while working, brief summary at the end — not play-by-plays.
- Runs sessions end-to-end: code → commit → PR → Jira update → deploy in a single flow.
- Uses `/wrapup` at session end; `/brief` at session start when resuming complex work.
- Does not use `/fast` mode by default.

**Communication:**
- No preamble, no filler. Get to the point.
- No emojis unless asked.
- Concise summaries beat comprehensive ones.
- Prefers to test things himself in the browser rather than have Claude narrate what it sees.
